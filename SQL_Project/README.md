# E-Commerce Data Insights using SQL Server

## Project Overview
This project focuses on analyzing e-commerce data using SQL Server to extract meaningful business insights.   
The goal is to understand sales performance, customer behavior, and product trends using SQL queries.

---

## Tools Used
- SQL Server
- SQL Server Management Studio (SSMS)

---

## Dataset Description
The dataset contains e-commerce related information such as:
- Orders
- Customers
- Products
- Sales amount
- Order dates

(Sample data is used for practice and analysis.)

---

## Key Insights Generated
- Total sales and revenue analysis  
- Monthly and yearly sales trends  
- Top-selling products  
- Top customers by revenue  
- Order count analysis  
- Customer purchase behavior  

---

## SQL Concepts Used
- SELECT statements  
- WHERE conditions  
- GROUP BY & HAVING  
- JOINS (INNER JOIN)  
- Aggregate functions (SUM, COUNT, AVG)  
- ORDER BY  
- Subqueries
